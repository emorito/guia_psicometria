Validez de Contenido

Se asegura que los ítems del test cubran representativamente el constructo. Se utiliza un panel de expertos para evaluar cada ítem y documentar los ajustes basados en su feedback.

Checklist de Validez de Contenido

Estudio Pre-Piloto

Es una prueba a pequeña escala para detectar problemas antes de la recolección principal. Se aplican entrevistas cognitivas o grupos focales para registrar el tiempo de respuesta y la claridad de las instrucciones.

Checklist del Estudio Pre-Piloto

Análisis de Ítems

Se examinan las propiedades estadísticas de cada pregunta. Se calcula la correlación ítem-total para identificar y decidir qué ítems se mantienen, revisan o eliminan para fortalecer el instrumento.

Checklist del Análisis de Ítems

Recolección Principal y Muestreo

Se pone a prueba el instrumento a gran escala. Las decisiones clave son: determinar un tamaño de muestra adecuado (N > 300), elegir un método de muestreo y seguir un estricto protocolo ético con consentimiento informado.

Checklist de Recolección y Muestreo

Análisis Psicométrico

Se procesan los datos recolectados. Se realiza una limpieza de la base de datos, se evalúa la estructura interna con Análisis Factorial Exploratorio (AFE), se analiza la fiabilidad y se crean los baremos para interpretar las puntuaciones.

Checklist del Análisis Psicométrico

Informe Final

El informe final resume y relata la historia completa del proyecto. Sigue la estructura IMRyD (Introducción, Método, Resultados y Discusión) y presenta los hallazgos, limitaciones y futuras aplicaciones del instrumento validado.

    Guía Práctica de Análisis en Jamovi

    Pasos esenciales para análisis psicométricos en Jamovi, con enlaces a manuales y tutoriales para profundizar y mantener la guía siempre actualizada.

    1. Configurar Jamovi

    • Descargue e instale la última versión desde la Página Oficial de Jamovi.
    • Consulte el Manual de Usuario Oficial para soporte general.

    2. Importar Datos

    • Video-Tutorial: Importar y gestionar datos en Jamovi (YouTube).
    • Pasos: Archivo → Abrir → Seleccionar archivo (.csv, .xlsx, .sav) → Verificar tipos de variable.

    3. Módulos Necesarios

    Para agregar módulos: pestaña Modules (icono de puzzle) → Jamovi Library → Install from Jamovi library.

    MóduloPropósito
    FactorAnálisis Factorial Exploratorio (EFA)
    jmv (Reliability)Coeficientes de fiabilidad (Alfa, Omega)
    lavaan (SEM)Análisis Factorial Confirmatorio (CFA)

    4. Análisis Factorial Exploratorio (EFA)

    • Ruta: Analyses → Factor → Exploratory Factor Analysis.
    • Guía de referencia: EFA in R and jamovi.

    5. Análisis de Fiabilidad

    • Ruta: Analyses → Factor → Reliability Analysis.
    • Lectura recomendada: Interpretación de Alfa y Omega.
