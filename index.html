Revisión Inicial y Plan

Esta es la fase de planificación del proyecto. Se responde a las preguntas clave: qué (se define el constructo), por qué (se justifica la necesidad de validación) y cómo (se crea un plan de trabajo detallado). El resultado es una ficha técnica y un cronograma que guiarán todo el proceso.

Validez de Contenido

En esta etapa se asegura que las preguntas del test cubran de forma representativa lo que se busca medir. Se utiliza un panel de expertos para evaluar cada ítem y asegurar que el instrumento sea una medida fiel del constructo, documentando todos los ajustes tras su retroalimentación.

Checklist de Validez de Contenido

Estudio Pre-Piloto

Es una prueba a pequeña escala para detectar problemas antes de la recolección principal. Se aplican entrevistas cognitivas o grupos focales a una muestra reducida para registrar el tiempo de respuesta y los comentarios de los participantes sobre la claridad de las instrucciones e ítems.

Checklist del Estudio Pre-Piloto

Análisis de Ítems

Se examinan las propiedades estadísticas de cada pregunta. Se calculan estadísticos descriptivos, como la correlación ítem-total, para identificar y marcar los ítems con un desempeño débil. El objetivo es decidir qué ítems se mantienen, revisan o eliminan para fortalecer el instrumento.

Checklist del Análisis de Ítems

Recolección Principal y Muestreo

Aquí se pone a prueba el instrumento a gran escala. Las decisiones más importantes son: determinar un tamaño de muestra adecuado (más de 300 participantes es una buena práctica), elegir un método de muestreo y seguir un estricto protocolo ético-logístico con consentimiento informado y autorización del comité de ética.

Checklist de Recolección y Muestreo

Análisis Psicométrico

En esta fase se procesan los datos. Primero se realiza una limpieza para manejar valores faltantes y atípicos. Luego se evalúa la estructura interna del test con Análisis Factorial Exploratorio (AFE), se analiza la fiabilidad (ej. Alfa de Cronbach, Omega) y, finalmente, se crean los baremos para estandarizar e interpretar las puntuaciones.

Checklist del Análisis Psicométrico

Informe Final

El informe final es el documento que resume y relata la historia completa del proyecto. Sigue la estructura IMRyD (Introducción, Método, Resultados y Discusión) y presenta de manera clara los hallazgos clave, las limitaciones y las futuras aplicaciones del instrumento validado.

Guía Práctica de Análisis en Jamovi

Pasos esenciales para análisis psicométricos en Jamovi, con enlaces a manuales y tutoriales para profundizar y mantener la guía siempre actualizada.

1. Configurar Jamovi

  • Descargue e instale la última versión desde la Página Oficial de Jamovi.
  • Consulte el Manual de Usuario Oficial para soporte general.

2. Importar Datos

  • Video-Tutorial: Importar y gestionar datos en Jamovi (YouTube).
  • Pasos: Archivo → Abrir → Seleccionar archivo (.csv, .xlsx, .sav) → Verificar tipos de variable.

3. Módulos Necesarios

Para agregar módulos: pestaña Modules (icono de puzzle) → Jamovi Library → Install from Jamovi library.

4. Análisis Factorial Exploratorio (EFA)

  • Ruta: Analyses → Factor → Exploratory Factor Analysis.
  • Guía de referencia: EFA in R and jamovi.

5. Análisis de Fiabilidad

  • Ruta: Analyses → Factor → Reliability Analysis.
  • Lectura recomendada: Interpretación de Alfa y Omega.
